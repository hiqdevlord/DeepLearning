The Graphics may be used in GGplot2 in our project.

Ggplot2 is a very powerful tool for data visualization.

Particularly, the followering graphics can be used in our final report, and we can also use some imagination to draw even more fancy graphs.

Scatter Plots

The data set uesd is in the package itself. We have different colors representing different years and also two stat_smooth lines for each year.

library(ggplot2)
p <- ggplot(mpg, aes(x = cty, y = hwy, colour = factor(year)))
p + geom_point() + stat_smooth()

## geom_smooth: method="auto" and size of largest group is <1000, so using
## loess. Use 'method = x' to change the smoothing method.

Also we can just have one stat_smooth line and add another factor.

You can also embed plots, for example:

p + geom_point(aes(colour = factor(year), size = displ)) + stat_smooth()

## geom_smooth: method="auto" and size of largest group is <1000, so using
## loess. Use 'method = x' to change the smoothing method.

We can use self-designed colors.

p + geom_point(aes(colour = factor(year), size = displ), alpha = 0.5, position = "jitter") + 
    stat_smooth() + scale_color_manual(values = c("blue2", "red4")) + scale_size_continuous(range = c(4, 
    10))

## geom_smooth: method="auto" and size of largest group is <1000, so using
## loess. Use 'method = x' to change the smoothing method.

And we can add one facet for different year


p + geom_point(aes(colour = class, size = displ), alpha = 0.5, position = "jitter") + 
    stat_smooth() + scale_size_continuous(range = c(4, 10)) + facet_wrap(~year, 
    ncol = 1)

## geom_smooth: method="auto" and size of largest group is <1000, so using
## loess. Use 'method = x' to change the smoothing method.

Bar chart

Base Plot

p <- ggplot(mpg, aes(x = class))
p + geom_bar()

Ordered by count

class2 <- mpg$class
class2 <- reorder(class2, class2, length)
mpg$class2 <- class2
P <- ggplot(mpg, aes(x = class2))
p + geom_bar(aes(fill = class2))

Different bar charts representing different year

p <- ggplot(mpg, aes(class2, fill = factor(year)))
p + geom_bar(position = "identity", alpha = 0.5)

Or like this in the stack.

p + geom_bar(position = "stack")

Two plots in the same graph

p + geom_bar(aes(fill = class2)) + facet_wrap(~year)

Or we can see their ratio, I think this may be helpful if we draw the graph to compare Taobao and Tmall

p + geom_bar(position = "fill")

Violin Plots

We can use violin plots to combine scatter plots also showing the different distributions.

ggplot(mpg, aes(class, hwy, fill = class)) + geom_violin(alpha = 0.3, width = 0.9) + 
    geom_jitter(shape = 21)
